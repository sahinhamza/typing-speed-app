## General info
This is a typing speed app. You can test how many words you can write in a minute.
	
## Technologies
Project is created with:
* JavaScript
* CSS
* react: 18.2.0
* redux: 4.2.0
* react-redux: 8.0.2
* reduxjs/toolkit: 1.8.5

	
## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm start
```
